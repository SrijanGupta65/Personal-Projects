Bridgeling Demo

AI assistant for institutional Q&A

Features

  • Multilingual: Ask questions in any language
  • Smart Retrieval: Finds answers from institutional content
  • Claude-Powered: Reasoning and generation via Claude 3.5 Sonnet
  • Embedded: Single-line widget embed, no setup required
  • Cited Sources: Links to original documents

Quick Start

Click the button below to initialize the widget. Try asking questions in any language!

Manual Initialization

Or initialize with custom settings:
